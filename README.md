# 烟记 SmokeTracker

一个纯本地的安卓抽烟 / 买烟记录 App。Kotlin + Jetpack Compose + Room，无需联网、无需账号，数据全部存在手机本地数据库。

## 功能

- **烟品库**：记录烟名、每包价格、每包支数、单根焦油量、尼古丁量；自动推导单根价格。第一支烟自动设为默认。
- **一键抽烟**：首页大圆按钮，抽一根点一次。按「当前默认烟品」记账，可随时从下拉切换。
- **今日实时统计**：当日根数、距上次抽烟时间间隔（每秒刷新）、当日花费、当日焦油 / 尼古丁摄入。
- **买烟记录**：按包记录购买，自动折算根数与花费；总价留空时按「包数 × 包价」估算。
- **库存联动**：买入根数 − 抽出根数 = 剩余库存，快抽完时提醒补货。
- **历史统计**：日 / 周 / 月 / 年 / 总计 五个维度汇总抽烟与买烟，并列出全部明细。
- **健康激励**：与昨天对比、本周日均、已记录天数。
- **近 7 天趋势图**：纯 Compose 柱状图，无第三方依赖。
- **误触撤销**：一键撤销上一根。

## 设计要点

- 抽烟事件保存「价格 / 焦油 / 尼古丁」**快照**，以后改烟价也不会篡改历史统计。
- 单一 `SmokeRepository` 数据入口 + `SmokeViewModel`，UI 用 `StateFlow` 驱动。

## 构建运行

1. 用 **Android Studio**（建议 Koala / 2024.1+）打开本工程根目录。
2. 首次打开会自动下载 Gradle Wrapper、依赖并同步。
3. 连接安卓真机或开模拟器，点 ▶ Run。

- minSdk 24，targetSdk / compileSdk 34
- Kotlin 2.0.20，Compose BOM 2024.09.02，Room 2.6.1

## 应用图标
